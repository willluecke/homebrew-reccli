# Homebrew RecCli

Official Homebrew tap for [RecCli](https://github.com/willluecke/reccli) - the dead-simple terminal recorder with a floating button.

## Install

```bash
brew install willluecke/reccli
```

Or:

```bash
brew tap willluecke/reccli
brew install reccli
```

## What is RecCli?

RecCli is a terminal recording tool that adds a floating record button to every Terminal window. One click to start, one click to stop. Perfect for:

- Debugging sessions
- Teaching and tutorials
- Documentation
- AI coding sessions (Claude Code, Cursor, etc.)
- Pair programming

## After Installation

The formula will guide you through setting up the auto-launch watcher. You can also run RecCli manually:

```bash
reccli watch     # Start background watcher
reccli launch    # Launch popup for current terminals
reccli status    # View recording stats
reccli gui       # Single popup mode
```
